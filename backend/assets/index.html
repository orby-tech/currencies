Currencies api service

About

This is a simple api service that provides currency exchange rates based on collection of apis.

Interface in Balances app

Api

Get all currencies (comming soon)

GET /api/currencies

This will return a list of all currencies with their respective exchange rates.

Get currencies latest rates

GET /api/rates-latest

This will return a list of currencies with their respective exchange rates.

Get currencies rates history

GET /api/rates-history

This will return a list of currencies with their respective exchange rates for the all period.
